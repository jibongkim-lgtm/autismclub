AI & CODING

AI Creator Coding

A 9-month integrated program where students build websites, games, and apps using Claude AI — designed for ages 11-18.

About the Program

AI Creator Coding is a hands-on project-based course where students use Claude AI to plan, design, and build real digital products. Over 9 months, participants progress through web development, game development, and app development — completing their own projects at each stage.

This program goes beyond simply asking AI questions. Students learn to leverage AI as a creative coding partner, developing practical skills in HTML, CSS, JavaScript, game logic, and app design while building a portfolio of original work.

Each Class Session

15 min Assignment Review — Review previous week's homework and share progress
20 min Concept & Prompts — Learn today's Claude AI prompts and coding concepts
60 min Hands-on Project — Build and create your own project with guidance
15 min Debug & Present — Fix errors and present your work to the class
10 min Homework — Receive next week's assignment and challenges
